# Metrics and Tracking

!!! info "Coming Soon"
    This tutorial is under development and will be available soon.

Learn how to extract metrics and build summaries from log data.

## What You'll Learn

- Track counts, sums, and averages
- Build custom metrics
- Use `--begin` and `--end` for initialization and finalization
- Display metrics with `--metrics`

## Prerequisites

- Completed the [Quickstart](../quickstart.md)
- Familiarity with Rhai scripting basics

## Overview

Topics covered:
- `track_count()`, `track_sum()`, `track_avg()` functions
- `track_unique()` for cardinality tracking
- `--metrics` flag for metric display
- Building dashboards from log metrics

*This tutorial will show you how to turn logs into actionable metrics.*
