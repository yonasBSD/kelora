# Find Errors in Logs

Learn how to quickly find and analyze errors across different log formats.

## Problem

You need to find error-level events in your logs, possibly across multiple files or formats, and understand what's happening around them.

## Solution

Topics covered:
- Level filtering with `--levels`
- Time-based filtering with `--since` and `--until`
- Context lines with `--before-context` and `--after-context`
- Combining filters for precise targeting

*This guide will show you how to find errors efficiently.*
